É NECESSÁRIO EFETUAR ALGUMAS CONFIGURAÇÕES ANTES DE IMPORTAR OU EXPORTAR SEU BANCO PELOS ARQUIVOS .BAT
 
- SE A VERSÃO DO SEU MYSQL FOR DIFERENTE DA VERSÃO 5.7 DEVE SER CRIADA UMA VARIAVEL DE AMBIENTE CHAMADA MYSQL_BIN E O SEU CONTEUDO É O CAMINHO PARA A PASTA BIN DO SEU MYSQL SERVER EX: C:\Program Files\MySQL\MySQL Server 8.0\bin

- CASO DÊ ERRO DE ENCODING AO IMPORTAR BANCO (utf8mb4) SEU BANCO DEVE SER SETADO PARA ESSE ENCODING ESPECIFICO ATRAVEZ DO COMANDO: ALTER DATABASE database_name CHARACTER SET = utf8mb4 COLLATE = utf8mb4_unicode_ci;

- ANTES DE IMPORTAR O BANCO VIA ARQUIVO DE BACKUP O MESMO DEVE SER CRIADO VAZIO EX: CREATE DATABASE EVENTDB
